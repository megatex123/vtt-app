## 1. Prepare Data

### Add audio files
Place `.wav` recordings in `voice-model/data/audio/`.  
Each file must be **‚â§ 30 seconds** (Whisper hard limit).

### Edit transcripts.csv
Add a row for each audio file:

```csv
filename,text
sample_001.wav,selamat pagi semua
sample_002.wav,terima kasih kerana hadir
```

### Install dependencies
```bash
cd /home/penyahpepijat/claude
pip install -r voice-model/requirements.txt
```

### Run preprocessing
```bash
python voice-model/src/prepare_data.py
```

Output:
- Resamples all audio to 16kHz mono
- Normalises amplitude to [-1, 1]
- Drops clips longer than 30s
- Saves HuggingFace DatasetDict to `data/processed/`
- Prints summary: total / train / val counts

---

## 2. Train Locally

Requires a CUDA GPU with at least 4GB VRAM for `whisper-small`.  
Use `whisper-base` in `config.py` for lower VRAM or CPU.

```bash
python voice-model/src/train.py
```

- Fine-tunes for 10 epochs (configurable in `src/config.py`)
- Saves best checkpoint by WER to `voice-model/models/`
- Prints final WER score

---

## 3. Train on Google Colab (Recommended)

Colab provides a free T4 GPU (~16GB VRAM).

### Step 1 ‚Äî Upload to Google Drive
Upload the entire `voice-model/` folder to your Google Drive.

### Step 2 ‚Äî Open the notebook
Open `colab_train.ipynb` in Google Colab:  
`File > Upload notebook` or open from Drive.

### Step 3 ‚Äî Set runtime
`Runtime > Change runtime type > T4 GPU`

### Step 4 ‚Äî Run all cells in order

| Cell | Action |
|------|--------|
| 1 | Check GPU (should show T4) |
| 2 | Install requirements |
| 3 | Mount Google Drive |
| 4 | Copy data to Colab local storage |
| 5 | Run prepare_data.py |
| 6 | Run train.py (~30‚Äì60 min) |
| 7 | Run evaluate.py, print WER |
| 8 | Save model back to Google Drive |
| 9 | Download model as .zip |

### Step 5 ‚Äî Copy model locally
Extract the downloaded zip to `voice-model/models/`.

---

## 4. Evaluate

```bash
python voice-model/src/evaluate.py
```

- Runs on validation split
- Prints per-sample WER table
- Saves aggregate results to `voice-model/results.json`

---

## 5. Run Inference

### Command line
```bash
python voice-model/src/inference.py --file path/to/audio.wav
```

Supported formats: `.wav`, `.mp3`, `.m4a`, `.flac`, `.ogg`, `.webm`

### Import in Python
```python
from voice_model.src.inference import transcribe

text = transcribe('audio.wav')
print(text)
```

### Fallback behaviour
- If `voice-model/models/` is empty ‚Üí uses `openai/whisper-small` base model
- If `voice-model/models/` has a checkpoint ‚Üí uses fine-tuned model
